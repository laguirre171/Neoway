O Arquivo neoway.py é a resposta do primeiro exercício. Para rodar ele é 
preciso baixar a biblioteca do Python Validate_email, usando o comando: pip install validate_email


O Arquivo neowaySoap.py é a resposta do segundo exercício. Para rodar ele é
preciso baixar a bilioteca httplib2, conforme orientação do site https://github.com/jcgregorio/httplib2
